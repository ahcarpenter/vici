    Replace with:
    ```python
    audit_repo = AuditLogRepository()
    handlers = [
        JobPostingHandler(
            job_repo=JobRepository(),
            audit_repo=audit_repo,
            pinecone_client=pinecone_client,
            extraction_service=extraction_service,
        ),
        WorkerGoalHandler(
            work_request_repo=WorkRequestRepository(),
            audit_repo=audit_repo,
        ),
        UnknownMessageHandler(
            twilio_client=twilio_client,
            extraction_service=extraction_service,
        ),
    ]
    orchestrator = PipelineOrchestrator(
        extraction_service=extraction_service,
        audit_repo=audit_repo,
        handlers=handlers,
    )
    ```

    Key facts verified from source:
    - JobRepository.create and WorkRequestRepository.create are instance methods (call
      self._persist), so JobRepository() and WorkRequestRepository() — instances, not classes.
    - AuditLogRepository is instantiated once and shared across all handlers and the
      orchestrator. This is correct: audit_repo is stateless (each .write() call receives
      a session), so sharing one instance is safe.
    - JobPostingHandler accepts: job_repo, audit_repo, pinecone_client, extraction_service
    - WorkerGoalHandler accepts: work_request_repo, audit_repo
    - UnknownMessageHandler accepts: twilio_client, extraction_service
    - PipelineOrchestrator accepts: extraction_service, audit_repo, handlers: list[MessageHandler]

    Also remove the now-unused imports from main.py if they are no longer referenced
    after this change:

      passed to the orchestrator; verify it is not used elsewhere in main.py before removing)
