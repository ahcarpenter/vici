## Problem

Renovate (automated dependency update bot) should be configured and integrated into the GitHub Actions workflow to keep dependencies up to date automatically.

## Solution

Set up Renovate configuration (renovate.json) and ensure the GitHub workflow supports it — e.g., auto-merge rules, PR labels, scheduling, and any required workflow permissions.
