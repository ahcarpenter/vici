# Phase 04 — Security

> Per-phase security contract: threat register, accepted risks, and audit trail.

---

## Trust Boundaries

| Boundary | Description | Data Crossing |
|----------|-------------|---------------|
| cluster-internal only | Jaeger, Prometheus, Grafana are ClusterIP — no external access (D-10) | Traces, metrics, dashboards (internal only) |
| OpenSearch unauthenticated | OpenSearch security disabled, cluster-internal only (Phase 3) | Trace storage reads/writes |
| GCP Secret Manager -> K8s Secret | ESO syncs OTEL endpoint; IAM controls access | Cluster-internal URL (low sensitivity) |
| grafana.com download | Temporal dashboard JSON fetched at Pulumi runtime | Public dashboard JSON (no secrets) |

| T-04-02 | Information Disclosure | Jaeger trace data in OpenSearch | accept | OpenSearch security disabled but cluster-internal only; acceptable for v1.0 single-tenant | closed |
| T-04-03 | Denial of Service | Jaeger collector OTLP endpoints | accept | Only reachable from within cluster; no public route; resource limits set | closed |
