## What Was Built

### src/pipeline/constants.py (new)
Pure constants module defining all OTel attribute keys used across pipeline, sms, and temporal domains. Eight constants covering messaging namespace (message_id, phone_hash, system, destination), database namespace (system, operation.name, vector.job_id), and application namespace (work_request.user_id).

### Span Coverage Added
- **pipeline.orchestrate** — wraps `PipelineOrchestrator.run()` body; child of `RunActivity:process_message_activity` TracingInterceptor span automatically; carries `messaging.message_id` and `messaging.source.phone_hash`
- **pipeline.handle_worker_goal** — wraps `WorkerGoalHandler.handle()` body; carries `messaging.message_id` and `app.work_request.user_id`
- **temporal.process_message** enriched — adds `messaging.message_id` and `messaging.source.phone_hash` to existing span
- **sms_webhook HTTP span enriched** — `router.py` calls `get_current_span().set_attribute()` on FastAPI auto-instrumented span with `messaging.message_id`, `messaging.source.phone_hash`, `messaging.system=twilio`

### PII Fix
`unknown.py`: replaced `span.set_attribute("messaging.destination", ctx.from_number)` (raw E.164 PII leak) with `span.set_attribute(OTEL_ATTR_MESSAGING_DESTINATION, hash_phone(ctx.from_number))`. Deprecated attribute name `messaging.destination` also corrected to `messaging.destination.name`.

### Semconv Fix
`job_posting.py`: replaced deprecated `db.operation` attribute with `db.operation.name` (OTel semantic conventions v1.24+).

## Test Coverage

- `test_process_message_emits_temporal_span`: extended to assert `messaging.message_id` and `messaging.source.phone_hash` present
- `test_orchestrator_emits_pipeline_span`: new — asserts `pipeline.orchestrate` span emitted with correct attrs
- `test_unknown_handler_span_uses_hashed_phone`: new — asserts `messaging.destination.name` is hashed (not raw E.164), deprecated `messaging.destination` absent
- `test_worker_goal_handler_emits_span`: new — asserts `pipeline.handle_worker_goal` span with `messaging.message_id` and `app.work_request.user_id`
- Existing `test_job_branch_emits_pinecone_span` and `test_unknown_branch_emits_twilio_span` updated to use new attribute names
